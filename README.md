# Dirty Money - Marked Bill Exchange Script for FiveM, QBCore
Configurable multi location marked bills exchange system with minigame

# Download
https://github.com/calilifeservers/cali_anchor

# Features
The Advanced Marked Bill Exchange Script includes the following features:

- Configurable Locations
- Configurable Metadata vs Amount
- Configurable Exchange Fee
- Configurable MiniGame Options
- Configurable Whitelist Options
- Configurable Discord Webhooks

# Instructions
Must have ps-ui or boii-minigames. Config to your liking.
